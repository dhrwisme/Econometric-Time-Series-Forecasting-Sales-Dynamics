# Econometric Time-Series Forecasting: Consumer Spending & Retail Dynamics

## 📌 Project Overview
This project involves a rigorous econometric analysis of consumer spending trends (1982-2025). The research focus is on benchmarking traditional statistical models (SARIMA, ETS) against modern forecasting requirements, specifically addressing long-term non-stationarity and complex seasonality.

## 🚀 Technical Highlights
- **Methodological Benchmarking:** Conducted a comparative study between **SARIMA** (Seasonal Auto-Regressive Integrated Moving Average) and **ETS** (Error-Trend-Seasonality) models.
- **Statistical Preprocessing:** Addressed **Heteroscedasticity** and non-stationary variance by implementing Box-Cox transformations and seasonal decomposition.
- **Model Validation:** Evaluated predictive performance using **RMSE** and the **Ljung-Box test** for residual autocorrelation.
- **Robustness:** Demonstrated that the **ETS(A,A,M)** model yielded superior robustness in capturing multiplicative seasonality within the retail sector.

## 🛠️ Tech Stack
- **Languages:** Python (Statsmodels, Pmdarima), R (for data extraction).
- **Libraries:** Pandas, NumPy, Scipy, Matplotlib, Seaborn.
- **Statistical Tests:** ADF (Augmented Dickey-Fuller), KPSS, Ljung-Box.
